# Faers-explorer
Exploring the FDA adverse-event (FAERS) dataset in Python — a pharmacology student's data project.
A learning project: exploring the FDA Adverse Event Reporting System (FAERS) — an open dataset of drug adverse-event reports — using Python.
I'm a first-year Pharmacology student at King's College London building my data skills in public. This repo tracks that, one small step at a time.
Goals

Load and clean real pharmacovigilance data
Explore which drugs and reactions appear most, and what that does (and doesn't) mean
Practise Python, pandas, and clear data storytelling

Status: just getting started 🚧
